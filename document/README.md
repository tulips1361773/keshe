# 乒乓球培训管理系统文档中心

## 项目概述

本项目是一个基于Django + Vue.js的乒乓球培训管理系统，旨在为乒乓球培训机构提供完整的学员管理、教练管理、课程预约、比赛组织等功能。

## 技术栈概览

### 后端技术
- **框架**：Django 4.x
- **数据库**：MySQL 8.0
- **缓存**：Redis
- **API**：Django REST Framework
- **认证**：JWT Token

### 前端技术
- **框架**：Vue.js 3.x
- **构建工具**：Vite
- **UI组件**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router

### 部署技术
- **容器化**：Docker
- **编排**：Docker Compose
- **Web服务器**：Nginx
- **应用服务器**：Gunicorn
