🏓 教练员注册功能测试

问题描述

用户注册教练员时遇到 "教练员必须填写比赛成绩描述" 错误，但注册页面没有提供填写成绩描述的字段。

修复方案
  • 在注册表单中添加教练员专用的成绩描述输入框
  • 只有当用户选择"教练"类型时才显示该字段
  • 添加相应的验证规则（必填，至少10个字符）
  • 在提交数据时包含achievements字段
测试用例 1: 教练员注册（有成绩描述）
{ "username": "coach123", "real_name": "张教练", "phone": "13800138000", "email": "coach@example.com", "password": "Abc12345!", "password_confirm": "Abc12345!", "user_type": "coach", "achievements": "曾获得全国乒乓球锦标赛冠军，有10年教学经验，培养过多名优秀学员。" }
测试用例 2: 教练员注册（无成绩描述）
{ "username": "coach456", "real_name": "李教练", "phone": "13800138001", "password": "Abc12345!", "password_confirm": "Abc12345!", "user_type": "coach" // 缺少 achievements 字段 }
测试用例 3: 学员注册（正常）
{ "username": "student123", "real_name": "王学员", "phone": "13800138002", "password": "Abc12345!", "password_confirm": "Abc12345!", "user_type": "student" // 学员不需要 achievements 字段 }
快速访问
注意：请在注册页面中选择"教练"类型，确认是否出现成绩描述输入框。
